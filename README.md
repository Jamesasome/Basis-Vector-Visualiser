# Basis-Vector-Visualiser
Using PyQt6 and pyqtgraph, plot a grid and two standard basis vectors, set new basis vectors and using slider visualise transformation of vectors and grid

### Features/Comments

- Plots a standard grid with two basis vectors in 2D, allows the input of two new basis vectors, and using slider animates or shows basis transformation of vectors and grids
- Inspired by [MatVis](https://shad.io/MatVis/), [3Blue1Brown](https://www.3blue1brown.com/), and [chanrt](https://github.com/chanrt)


### Dependencies
- Numpy
- pyqtgraph
- PyQt6
